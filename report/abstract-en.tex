\begin{abstract}
	\noindent
	This is a report about a system in PHP built on the framework Laravel, with
	MySQL as a database. It is a prototype for a large system and has a basic
	user system as well as functionality to post and view messages. It has been built
	to minimise the most common security risks.

	\keywords{Laravel, Web Application Security, PHP, MySQL}
\end{abstract}
